# Module Images Support

WordPress image enhancement module with filters, social optimization, text overlays, branding features, and composition guides.

## Features

### Composition Guides
Visual guides to assist with image composition during manual cropping:

- **Rule of Thirds**: Classic 3x3 grid overlay showing optimal focal points
- **Golden Ratio**: Fibonacci spiral overlay based on the golden ratio (φ ≈ 1.618)
- **Interactive Controls**: Toggle grid visibility, select grid type, adjust opacity
- **Real-time Preview**: Instant overlay updates during image editing

## Installation

1. Upload the plugin files to `/wp-content/plugins/module-images-support/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Media Library to access composition guides during image editing

## Usage

1. Open WordPress Media Library
2. Select an image to edit
3. Composition Guides panel appears in the editing interface
4. Enable grid overlay and select your preferred guide type
5. Adjust opacity to suit your needs
6. Use the visual guide to improve image composition

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- PHP GD Library enabled
- Modern browser with HTML5 Canvas support
